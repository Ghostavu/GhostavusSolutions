# Ghostavus Solutions

## Status do Projeto
Em desenvolvimento

## Tecnologias Utilizadas
- Java
- MySQL
- JDBC
- NetBeans

## Time de Desenvolvedores
- Gustavo Kisner

## Objetivo do Software
O sistema Ghostavus Solutions é um software desktop desenvolvido para auxiliar na gestão de vendas, clientes, fornecedores e controle de estoque.

## Funcionalidades do Sistema
- Cadastro de usuários
- Cadastro de clientes
- Cadastro de fornecedores
- Cadastro de produtos
- Registro de pedidos
- Controle de estoque
- Consulta de vendas
