Sistema desktop desenvolvido em Java para gestão de clientes, fornecedores, produtos, pedidos e controle de estoque.

## 🎯 Objetivo do Software

O objetivo do sistema **Ghostavus Solutions** é fornecer uma solução desktop para gerenciamento de processos comerciais, permitindo o controle de clientes, fornecedores, produtos e vendas de forma organizada e eficiente.

---

## ⚙️ Funcionalidades do Sistema

O sistema possui as seguintes funcionalidades:

- Cadastro de usuários
- Cadastro de clientes
- Cadastro de fornecedores
- Cadastro de produtos
- Registro de pedidos
- Controle de estoque
- Consulta de vendas
- Integração com banco de dados MySQL

---

## 🗄️ Estrutura do Projeto

O projeto segue uma organização baseada em camadas:

- **DAO** → comunicação com o banco de dados  
- **Model** → representação das entidades do sistema  
- **View** → interface gráfica do sistema  
- **Util** → classes auxiliares

---

## 🛠️ Como Executar o Projeto

1. Instalar o Java JDK
2. Instalar o NetBeans
3. Instalar o MySQL
4. Importar o banco de dados do sistema
5. Abrir o projeto no NetBeans
6. Executar a aplicação
